## 🛠 Tech Stack

- [Three.js r128](https://threejs.org/) — WebGL 3D library
- `ACESFilmicToneMapping` + `sRGBEncoding`
- `AdditiveBlending` for glow and light layering
- `IcosahedronGeometry` + `OctahedronGeometry` for crystal rendering
- Pure HTML / CSS / JavaScript — no build tools required
